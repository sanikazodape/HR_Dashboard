# 📊 HR Analytics Dashboard

A modern Power BI dashboard designed to analyze workforce demographics, track employee retention, and optimize HR operations.

## 🚀 Key Insights & Features
* **Employee Attrition Analysis**: Tracks turnover rates by department, role, and tenure.
* **Demographics Overview**: Displays workforce distribution across gender, age brackets, and locations.
* **Performance vs. Compensation**: Correlates salary bands with employee performance ratings.

## 🛠️ Tech Stack Used
* **Business Intelligence**: Power BI Desktop (Project Mode - `.pbip`)
* **Data Sources**: CSV / Excel (Mock HR Data)
* **Languages**: DAX (Data Analysis Expressions) for custom metrics
